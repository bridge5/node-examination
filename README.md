# node-examination

Building a RESTful CRUD API with Node.js(>=v10.15.0), Express/Koa and MongoDB/MySQL.

## Usage

```
// API documentation
$ git clone git@github.com:bridge5/node-examination.git
$ cd node-examination/swagger
$ npm start

$ open http://localhost:3030
```
